Finding Sunny: A Web Adventure

Welcome to Finding Sunny, an interactive game where the player embarks on a quest to locate a missing dog named Sunny. As you explore, solve puzzles, and interact with characters, you will uncover clues to Sunny's whereabouts.

Game Overview

This game features various levels, each with different mini-games and puzzles. Players must make decisions that affect the outcome of the story. Are you ready for the challenge?

  • Explore different locations like the candy stand, park, and market.
  • Interact with characters to gather clues.
  • Solve puzzles to unlock new areas.

How to Play

Use the buttons and input fields to interact with the game. Each choice you make will affect the storyline. Can you find Sunny?
